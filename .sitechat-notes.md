# Site Notes

- 2026-07-22: `js/custom.js` had leftover code (copy-pasted from another client's theme, "ISS") that forced the navbar logo image `src` to `https://iss.buzzinga.co/...iss-logo.svg` / `iss-logo-white.svg` on page load and on every scroll tick (was around lines 250-284). Since those URLs don't exist on this site, the logo showed as a broken/placeholder image whenever the visitor scrolled, and only looked right right after a fresh page load (before the scroll handler ran). Removed the logo-src-swapping code entirely so the logo Elementor renders normally is left alone; kept the harmless "active section" scroll-highlight logic that was in the same handler. If a future request wants a genuine light/dark logo swap on scroll, it needs to be built fresh using this site's real logo assets, not copied from another client site.

- 2026-07-22: File edits (via the Edit tool) get auto-committed to a `change/chg_*` git branch in the repo, and a separate automatic process later merges + "publishes" + deploys them to the live preview (visible as "Publish change" and "deploy marker" commits appearing on their own). This publish step is NOT instant - it can take several minutes, sometimes longer. Manually running `git merge`/`git push` to main was tried once to force it through and did not obviously speed things up, so it's better to just make the file edit and wait/re-check with check_page or check_style than to fight the git state directly.
